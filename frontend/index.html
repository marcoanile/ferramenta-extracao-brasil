🔐 Configurar Credenciais TOConline
Como obter as credenciais:
1. Aceda ao TOConline com conta de Empresário
2. Menu → Empresa → Configurações → Dados API
3. Introduza o nome e email do integrador
4. O integrador receberá um e-mail com Client ID, Secret e OAUTH_URL

Credenciais configuradas. Clique em Iniciar Autenticação para abrir a janela de login TOConline.
Após autorizar no browser, a página será atualizada automaticamente.

Redirect URI registado:
http://localhost:5000/api/auth/toconline/callback
Este URL deve estar registado nos Dados API do TOConline
